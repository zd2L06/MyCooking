# 概要
料理を自由に投稿できるサービスです。 

子供から大人まで幅広い利用者を想定しています。

## ターゲット
- 小学生
- 中学生
- 高校生
- 大学生
- 社会人

## 使い方
1. ユーザー登録
1. メール送信
1. 写真投稿
1. 承認ボタン

## ページ一覧
| URL | サーブレット | JSP |
|:---:|:---|---:|
| /home | HomeServlet | home.jsp |
| /top | TopServlet | top.jsp |
